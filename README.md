# MVP-and-pet-projects-MIPT
Система автоматической сегментации и профилирования клиентов (MVP)
Бизнес-контекст
Переход от массовых рассылок к персональным стратегиям для крупного ритейла.
Проблема: Высокие затраты на нерелевантные промо-акции и отсутствие разметки базы.
Решение: ML-сервис для выделения устойчивых потребительских кластеров и классификации новых пользователей в реальном времени.
Ключевые результаты
Экономический эффект: Прогнозный чистый Uplift прибыли ~17.9 млн руб./год на анализируемую базу (при марже 25%).
Качество модели: F1-score (Macro) = 0.95 на отложенной тестовой выборке.
Масштаб: Обработан массив из 4.2 млн транзакций (анонимизированные данные реальной сети).
Технологический стек
Моделирование: CatBoostClassifier, K-Means (кластеризация), Isolation Forest (очистка выбросов).
Feature Engineering: Создано 38 признаков (RFM-метрики, доли 25 категорий товаров, предпочтения по времени суток и типу покупки).
Анализ: PCA (визуализация 2D/3D), классический EDA, Feature Importance.
Deployment: Реализован прототип интерфейса на Streamlit для скоринга новых клиентов.
Roadmap развития проекта
Time Series Split: Переход к валидации на временных рядах для исключения риска «заглядывания в будущее»
Точка стабилизации: Расчет минимального порога данных (например, 3-я покупка) для старта персонализации
RecSys: Внедрение алгоритмов ассоциативных правил (FP-Growth/Apriori) для кросс-продаж внутри сегментов
Анализ миграции: Расчет матриц переходов клиентов между кластерами для отслеживания смены лайфстайла
